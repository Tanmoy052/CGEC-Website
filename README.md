# Cooch Behar Government Engineering College (CGEC) Website

This is a full-stack website for CGEC, featuring a modern UI/UX, role-based access, admin dashboard, and AI chatbot.

## Project Structure

- `frontend/`: Next.js frontend application.
- `backend/`: Node.js/Express backend API.

## Getting Started

### Prerequisites

- Node.js (v18+)
- MongoDB

### Installation

1. Clone the repository.
2. Install dependencies for both frontend and backend:
   ```bash
   npm install
   ```

### Running the Project

#### Backend
```bash
cd backend
npm run dev
```

#### Frontend
```bash
cd frontend
npm run dev
```

## Features

- **Modern UI/UX**: Built with Next.js, Tailwind CSS, and Framer Motion.
- **Role-based Access**: Separate portals for Students, Faculty, and Admin.
- **AI Chatbot**: Intelligent assistant for college-related queries.
- **Notice Board**: Categorized notices (Notice, Tender, News, Recruitment).
- **Department Pages**: Dedicated sections for CSE, ECE, EE, ME, and Civil Engineering.
